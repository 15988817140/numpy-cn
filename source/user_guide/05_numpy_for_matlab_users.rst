==================================
Numpy对于Matlab用户
==================================

----------------------------------
介绍
----------------------------------

MATLAB®和NumPy / SciPy有许多共同之处。 但是有很多不同之处。 创建NumPy和SciPy是为了用Python以最自然的方式进行数值和科学计算，而不是MATLAB®的克隆。 本页面旨在收集两者差异的智慧，主要目的是帮助熟练的MATLAB®用户成为精通NumPy和SciPy的用户。
